I would be more than happy to! One thing I first want to be clear of is which model, of the few different versions of this static cat hindlimb model, you would like to use.

Below are list of the two different versions of the model and brief description of their features:
1) models used in McKay and Ting, 2008 to compute FFS:
- static models for 3 cats
- posture matched to kinematic data of 3 or 4 configurations in each cat
- model maps muscle excitation to endpoint force, using pseudo-inverse of Jacobian transpose
 
2) models used in Sohn et al., 2013 to compute Feasible Ranges
- only one of the postures from McKay and Ting, 2008 was used for the same 3 cats
- model maps muscle excitation to joint torque
- moment arm matrix, endpoint Jacobian, and scaling factors for active muscle force generation were derived from a dynamic model implemented in software Neuomechanic (Bunderson et al., 2012), and is slightly different from those used in McKay and Ting, 2008

Also, do you want just the models so that you will be specifying your own tasks (e.g. endpoint force)?
Or do you also want the experimental force data that we used? We have five experimentally-derived force vectors in each cat at each posture from Torres-Oviedo et al., 2006.
